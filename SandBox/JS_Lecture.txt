Lecture6 구문과 표현식

//expression
console.log(doA);

//statement
function doA() {

}

-------------------------

//expression
var doA = function() {

};

구문(statements) : 코드의 해석에 필요한 틀 -> 선언문
표현식(expression) : 결과를 만들어 내는 코드의 조각 -> 연산, () 실행식

표현식들이 모여 구문 이룬다.
해석단계는 구문 처리
실행할때는 식 처리

example)

//statement
var x;
if(y >= 0) {
	x = y;
} else {
	x = -y;
}

//expression
var x = (y >= 0 ? y : -y);	-> 해석될 동작하는게 아니라 실행할 때 동작
				메모리만 차지하는 상태, undefined
